# CCC ligand-receptor priors

`ligand_receptor_human.tsv` is the preferred pipeline input. It is intentionally generated by `notebooks/00_download_grn_prior.ipynb` or:

```bash
python scripts/download_ccc_prior.py --outdir resources/ccc_prior
```

The downloader attempts to normalize and merge OmniPath, CellPhoneDB and CellChatDB human ligand-receptor resources.

`ligand_receptor_seed_crc.tsv` is only a small smoke-test fallback retained from the earlier patch. Do not describe it as a full database prior in reports or manuscripts.
